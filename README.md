#  [![NPM version][npm-image]][npm-url] [![Dependency Status][daviddm-image]][daviddm-url]

> Unify JSON documents


## Install

```sh
$ npm install --save unionj
```


## Usage

```js
var unionj = require('unionj');

var result = unionj.add(json1, json2);
```
